# ☕ Java: Criando a Sua Primeira Aplicação

> Curso da plataforma [Alura](https://cursos.alura.com.br/course/java-criando-primeira-aplicacao) — Back-End | Nível Iniciante

---

## 📋 Sobre o Curso

Este curso é voltado para quem está dando os **primeiros passos em Java**, especialmente pessoas que já possuem algum conhecimento de lógica de programação e desejam ingressar no desenvolvimento **Back-End** com Java.

A proposta é guiar o aluno desde a configuração do ambiente até a criação de uma aplicação funcional, cobrindo os fundamentos essenciais da linguagem e da plataforma Java.

- ⏱️ **Carga horária:** ~8 horas
- 🎯 **Nível:** Iniciante
- 🏫 **Plataforma:** Alura
- 📜 **Certificado:** Sim (reconhecido pelo mercado)

---

## 🎯 Objetivos do Curso

Ao concluir este curso, serei capaz de:

- ✅ Configurar o ambiente de desenvolvimento Java (JDK + IntelliJ IDEA)
- ✅ Entender a estrutura básica de um programa Java
- ✅ Compreender o processo de compilação e execução (bytecode + JVM)
- ✅ Trabalhar com os principais **tipos de dados** em Java
- ✅ Realizar **leitura de dados** via console com a classe `Scanner`
- ✅ Aplicar **controle de fluxo** com `if/else`, `switch`, `while` e `for`
- ✅ Construir uma aplicação prática: um **catálogo de filmes**

---

## 📚 Conteúdo e Módulos

### 1. 🛠️ Preparando o Ambiente
- Instalação do **JDK** (Java Development Kit)
- Instalação e configuração da IDE **IntelliJ IDEA**
- Criação do primeiro projeto Java

### 2. 💡 O Primeiro Programa
- Estrutura de um programa Java (`public class`, `main`, etc.)
- Escrevendo o clássico "Hello, World!"
- Compilando e executando o programa

### 3. 🔍 Desvendando o Arquivo `.class`
- O que é compilação em Java
- Geração de **bytecode**
- Como a **JVM (Java Virtual Machine)** executa o programa

### 4. 🎬 Construindo o Catálogo de Filmes
- Criação de uma aplicação prática
- Declaração e manipulação de **variáveis**
- Entrada de dados com a classe `Scanner`
- Exibição e formatação de resultados no console

### 5. 🔄 Controlando o Fluxo da Aplicação
- Condicionais: `if`, `else if`, `else`
- Seleção múltipla com `switch`
- Laços de repetição: `while` e `for`
- Lógica de controle aplicada ao catálogo de filmes

### 6. 🏋️ Consolidando Conhecimentos
- Exercícios práticos para fixação
- Desafios para reforçar os conceitos aprendidos

## ✅ Classes já Implementadas

- `camada sessão 1/src/Perfil.java`: imprime duas mensagens no console:
  - `Ola, Leandro!`
  - `Tudo bem?`
- `camada sessão 1/src/Estudos.java`: imprime anotações de estudo no console
- `camada sessão 1/src/Soma.java`: imprime o resultado de `10 + 5` (saída: `15`)
- `camada sessão 1/src/Subtracao.java`: imprime o resultado de `10 - 5` (saída: `5`)
- `camada sessão 2/src/Media.java`: calcula a média de duas notas decimais
- `camada sessão 2/src/Casting.java`: converte um valor `double` para `int` com casting explícito
- `camada sessão 2/src/Palavras.java`: concatena uma variável `char` com uma variável `String`
- `camada sessão 2/src/Produto.java`: calcula valor total com preço e quantidade
- `camada sessão 2/src/ConversaoDolaresReais.java`: converte dólar para real com cotação fixa
- `camada sessão 2/src/CalculoDesconto.java`: calcula novo preço com desconto percentual
- `camada sessão 3/src/SinalDoNumero.java`: classifica um número como positivo ou negativo
- `camada sessão 3/src/QuemEOMaior.java`: compara dois inteiros e informa a relação entre eles
- `camada sessão 3/src/AreaEmCena.java`: calcula área de quadrado ou círculo via menu
- `camada sessão 3/src/TabuadaDoDia.java`: imprime a tabuada de 1 a 10
- `camada sessão 3/src/ParOuImpar.java`: verifica se o número é par ou ímpar
- `camada sessão 3/src/FatorialEmAcao.java`: calcula o fatorial usando estrutura de repetição

## 🚀 Como Executar o Projeto

### Pré-requisitos
- [JDK 17+](https://www.oracle.com/java/technologies/downloads/) instalado
- [IntelliJ IDEA](https://www.jetbrains.com/idea/download/) (ou outra IDE de sua preferência)

### Passos
```powershell
# 1. Clone o repositório
git clone https://github.com/Moquiuti/Java-criando-a-sua-primeira-aplica-o.git

# 2. Abra o projeto na IntelliJ IDEA

# 3. Execute a camada sessão 1
javac (Get-ChildItem "camada sessão 1\src\*.java").FullName
java -cp "camada sessão 1\src" Perfil
java -cp "camada sessão 1\src" Estudos
java -cp "camada sessão 1\src" Soma
java -cp "camada sessão 1\src" Subtracao

# 4. Execute a camada sessão 2
javac (Get-ChildItem "camada sessão 2\src\*.java").FullName
java -cp "camada sessão 2\src" Media
java -cp "camada sessão 2\src" Casting

java -cp "camada sessão 2\src" Produto
java -cp "camada sessão 2\src" ConversaoDolaresReais
java -cp "camada sessão 2\src" CalculoDesconto

# 5. Execute a camada sessão 3
```
