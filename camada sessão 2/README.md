# Camada Sessao 2

Exercicios praticos sobre variaveis, tipos primitivos, String e casting em Java.

## Classes implementadas

- `src/Media.java`
- `src/Casting.java`
- `src/Palavras.java`
- `src/Produto.java`
- `src/ConversaoDolaresReais.java`
- `src/CalculoDesconto.java`

## Como executar (PowerShell)

```powershell
javac (Get-ChildItem "src\\*.java").FullName
java -cp src Media
java -cp src Casting
java -cp src Palavras
java -cp src Produto
java -cp src ConversaoDolaresReais
java -cp src CalculoDesconto
```
